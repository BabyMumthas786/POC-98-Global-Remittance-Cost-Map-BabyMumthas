# Global Remittance Cost Map (POC-98)

A production-grade intelligence dashboard for the **Real Rails Intelligence Library** comparing remittance costs, settlement speed, accessibility, and corridor-level characteristics across major global routes.

This platform helps users understand how money moves internationally, who controls the underlying payment rails, and the trade-offs between formal and informal transfer systems.

## Features

- **Corridor Map**: Interactive geospatial visualization showing global flows, country nodes, and detailed routing overlays.
- **Fee Compare**: Deep comparison of provider costs, fixed fees, and FX spread margins using interactive charts.
- **Speed Ladder**: Direct relative visualization of provider settlement times (Instant vs. Same Day vs. Multi-day).
- **Access Points**: Breakdown of access methods and regional accessibility scores.
- **Corridor Explorer**: Institutional-grade searchable and filterable database of corridors with CSV exporting capabilities.
- **Payment Rails Education**: Rich details explaining the roles of Central Banks, Card Networks, SWIFT, fintechs, and Hawala networks.
- **Download Center**: Exposes raw datasets (CSV and JSON) generated by the ETL service.

## Tech Stack

- **Frontend**: Next.js 15+, TypeScript, Tailwind CSS, React Leaflet, Recharts, TanStack Table, Lucide React.
- **Backend**: FastAPI, Python, Pandas, Pydantic.

## Running Locally

### Backend Setup
1. Open a terminal and navigate to `/backend`
2. Create and activate a Python virtual environment:
   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Start the FastAPI server:
   ```bash
   python run.py
   ```
   The backend will run on `http://localhost:8000`.

### Frontend Setup
1. Open a terminal and navigate to `/frontend`
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
   The application will run on `http://localhost:3000`.
